## Installation

Vous pouvez installer le package via composer :

```bash
composer require dimtrovich/blitzphp-htmx
```

## Utilisation

```php
$skeleton = new Dimtrovich\BlitzPHP\Htmx();
echo $skeleton->echoPhrase('Hello, Dimtrovich!');
```
